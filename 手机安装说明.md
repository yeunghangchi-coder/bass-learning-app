# 手机安装说明

这个文件夹已经是一个手机可安装的网页 app。

## 现在为什么还不能直接装

你现在打开的是本机文件地址：

`file:///Users/yeunghangchi/Documents/学习/index.html`

手机安装需要一个网页地址。把这个文件夹放到网页空间后，用手机浏览器打开那个地址，就可以添加到主屏幕。

## iPhone

1. 用 Safari 打开 app 的网页地址。
2. 点底部分享按钮。
3. 选择“添加到主屏幕”。
4. 点“添加”。

## Android

1. 用 Chrome 打开 app 的网页地址。
2. 点右上角菜单。
3. 选择“安装应用”或“添加到主屏幕”。
4. 点“安装”。

## 需要上传的文件

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `sw.js`
- `icon.svg`
