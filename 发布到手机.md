# 发布到 iPhone 使用

这个 app 已经整理成静态网页 app。iPhone 不能直接安装电脑里的 `file://` 文件，需要先发布成一个网页地址。

## 最简单方式：Netlify 拖拽发布

1. 打开 Netlify 的网站。
2. 登录或注册。
3. 找到“Add new site”或“Deploy manually”。
4. 把整个 `学习` 文件夹，或 `bass-learning-app.zip` 上传。
5. 发布完成后，Netlify 会给你一个网址。

## 在 iPhone 上添加到主屏幕

1. 用 iPhone 的 Safari 打开发布后的网址。
2. 点底部分享按钮。
3. 选择“添加到主屏幕”。
4. 名字可以填“贝斯记录”。
5. 点“添加”。

以后就可以从 iPhone 主屏幕直接打开。

## 上传文件清单

上传时至少要包含这些文件：

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `sw.js`
- `icon.svg`
